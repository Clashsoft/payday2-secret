How to Use

Before beginning, consult the general steps outlined in this guide. This tool serves as an alternative to Ctrl-F on the GitHub page.

Normal Mode

Enter the translation of the symbols in the text field. The more letters you type, the further you can narrow down the list of achievements. When only one is left, you can stop translating the symbols.

Three-Symbol Mode

This is a more sophisticated way of translating the riddles. Instead of starting at the front, you only have to translate three symbols. Those are:

  • The fifth from the top right.
  • The sixth from the bottom left.
  • The second from the bottom left.

Shortcuts

Press Enter to pin the first result and Shift+Enter to mark it as completed.
